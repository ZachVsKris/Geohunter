# Geo: Second Coming — v10.5 Compact Mobile Board

World Bank 76 experiment built from the known-fast v9 baseline.

- 76 World Bank categories
- Maximum two categories per subject family
- Single-screen mobile drag-and-drop board
- Ten-country bank in two compact rows
- Eight category targets in a two-column grid
- Full category descriptions remain visible
- Short mobile country labels prevent vertical word wrapping
- Native mobile sharing with a cleaner spoiler-free score summary
- Night-atlas visual system retained
- No external fonts, images, or additional data requests
